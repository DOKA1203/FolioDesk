## 📖 소개

FolioDesk는 iOS / Android의 앱 폴더 경험을 Windows 바탕화면으로 가져옵니다.

바탕화면에 흩어진 아이콘들을 모바일처럼 폴더로 묶어 정리하고, 클릭 한 번으로 열고 닫을 수 있습니다. 복잡한 설정 없이 드래그앤드롭으로 바로 사용할 수 있어요.

## ✨ 기능

- 📁 **모바일 스타일 앱 폴더** — iOS / Android와 동일한 방식으로 앱을 폴더에 묶어 정리
- 🖱️ **드래그앤드롭** — 아이콘을 끌어다 폴더에 넣거나 꺼내기
- ⚡ **가벼운 실행** — 백그라운드 리소스 없음.

## 🚀 시작하기

1. [최신 릴리즈](https://github.com/yourname/foliodesk/releases/latest)에서 `FolioDesk-Setup.exe`를 다운로드합니다.
2. 설치 후 FolioDesk를 실행해 폴더를 생성합니다.
3. 바탕화면에서 아이콘을 드래그해 폴더를 만들어보세요.
